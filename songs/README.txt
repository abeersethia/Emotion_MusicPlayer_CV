SONG FOLDER STRUCTURE
=====================

Please add your audio files to the following folders:

📁 happy/    - Add upbeat, cheerful songs here
📁 sad/      - Add melancholic, sad songs here  
📁 neutral/  - Add calm, neutral songs here

Supported formats: MP3, WAV, OGG, FLAC

Example:
--------
songs/
├── happy/
│   ├── happy_song_1.mp3
│   ├── happy_song_2.mp3
│   └── upbeat_track.mp3
├── sad/
│   ├── sad_song_1.mp3
│   ├── melancholic_track.mp3
│   └── emotional_ballad.mp3
└── neutral/
    ├── calm_music.mp3
    ├── ambient_track.mp3
    └── relaxing_song.mp3

The program will randomly select songs from the appropriate folder
based on your detected emotion.
